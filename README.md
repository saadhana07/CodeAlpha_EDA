📊 Exploratory Data Analysis (EDA) Project

📌 Overview

This project performs Exploratory Data Analysis (EDA) on the Tips dataset to understand patterns, relationships, and key insights in customer spending and tipping behavior.

---

🎯 Objectives

- Analyze the structure of the dataset
- Identify trends and patterns
- Visualize relationships between variables
- Detect outliers and anomalies
- Generate meaningful insights

---

🗂️ Dataset

- Name: Tips Dataset
- Source: Built-in dataset from Seaborn library
- Description: Contains information about restaurant bills, tips, gender, smoking status, day, time, and group size

---

🛠️ Tools & Technologies

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook (Anaconda)

---

📊 Analysis Performed

- Data loading and inspection
- Statistical summary using describe()
- Missing value check
- Data visualization using:
  - Bar Plot
  - Scatter Plot (with color coding)
  - Regression Plot
  - Box Plot

---

📈 Key Insights

- Higher total bill leads to higher tips
- Dinner time generates higher revenue than lunch
- Spending varies across different days
- Some outliers are present in total bill

---

📷 Sample Visualizations

Graphs include:

- Total Bill by Day
- Bill vs Tip Relationship
- Outlier Detection
- Lunch vs Dinner Comparison

---

📌 Conclusion

The analysis shows clear relationships between customer spending and tipping behavior. Visualization techniques helped in identifying patterns, trends, and anomalies effectively.
